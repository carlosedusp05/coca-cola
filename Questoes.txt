O que é um repositório no Git e qual a sua importância?
É meio que uma pasta virtual, onde você salva seus projetos. Já a sua impotância é dada pelo poder de ter acesso a esse arquivo de qualquer lugar do mundode acesso à internet.

Qual a função do Git Bash?
É um aplicativo no windows que permite o git através de comandos.

O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
Ele adiciona os arquivos. A diferença do ponto para o sem ponto é que um adiciona tudo outro um arquivo em específico.

Por que utilizamos o ponto (.) no comando git add .?
Para nos referir a tudo "all".

Explique o comando git commit e por que é importante criar uma mensagem descritiva.
Ele é um salvamento que pode ser descrevido para podermos nos localizar em nosso projeto.

Qual a função do comando git push?
Serve para enviar alterações no arquivo.

Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
É importante para uma melhor localização do projeto e seus salvamentos.

Explique a função das tags HTML e como elas estruturam uma página.
Ela temcomo função de um esqueleto no corpo humano, ela estrutura o projeto.

O que são atributos em uma tag HTML e qual a sua função?
São os que completam as tags para estruturar e separar as funções da página.

O que é o Emmet e como ele facilita o desenvolvimento de páginas HTML?
O emmet é um facilitador de escrita de códigos que é usado para completar os códigos, aonde nele você escreve o codigo e ele completa ao apertar "enter/return".

O que é o box-sizing no CSS e por que usamos ele no reset inicial?
O box sizing é uma função onde denomina o tamanho da página que ela ocupará, é feita no começ pois as demais funções dependem dela para melhor layout.

Por que é importante utilizar tags semânticas no HTML?
Para estruturarmos e separar o corpo da nossa página como por exemplo: main, footer e header.

O que são media queries no CSS e qual a sua função?
Ele se adpta ao tamanho do sipositivo para a página ficar proporcional.

Qual a função do GitHub Pages?
É lançar sua página no ar da internet, para visualizar.

O que é o Flexbox no CSS e para que ele é utilizado?
O flexbox é um jeito onde organiza os elementos dividos por áreas.

Qual a diferença entre justify-content e align-items em um container Flexbox?
A diferença é que um alinha na vertical reta y e o outro na horizontal reta x.

Como o flex-grow funciona e qual o seu propósito no layout Flexbox?
Ele define o quanto de espaço o elemento vai ocupar dentro do container usando o display flex.

Explique a diferença entre flex-direction: column; e flex-direction: row;.
O column faz a divisão em colunas e o outro em linhas.
